# Ejecutar en desarrollo

1. Clonar el repositorio
2. Ejecular 
```
npm i
```
3. Tener Nest CLI instalado, para el desarollo
```
npm i -g @nestjs/cli
```
4. levantar base de datos 
```
docker-compose up -d
```

## Stack usado
- Mongo
- Nest
